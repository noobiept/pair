# [Try it out](https://nbpt.eu/games/pair/)

Match identical tiles with each other, to make a pair. Find all the pairs to win the game.

![Screenshot](screenshot.png)

# Development

- `corepack enable`
- `pnpm install`
- `pnpm start`
- `pnpm run build`
